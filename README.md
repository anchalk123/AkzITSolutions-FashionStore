# Fashion Store
E-commerce website UI/UX Project.
A modern, fully responsive fashion e-commerce website frontend built as an internship/acedemic project. Clean UI, smooth UX and complete shopping flow from homepage to checkout

---

## 🔗 Project Links
* **Live Demo:** https://akzitsolutions-fashionstore.onrender.com
* **Github Repo:** https://github.com/anchalk123/AkzITSolutions-FashionStore/tree/main

---

## ✨ Features
* **Multi-Page Layout:** Pages include Home, Collections, New Arrivals, Deals, and About.
* **Category Filters:** Sidebar links to filter products instantly (Women, Men, Home, etc.) without page reload.
* **Shopping Cart:** Live dynamic cart count badge syncing on all headers.
* **Local Storage:** Auto-saves cart products so data is safe even after refreshing the page.
* **Responsive UI:** Clean grid system optimized for both mobile and desktop views.

---

## 🛠️ Tools & Tech Used
* **HTML5:** Semantic components for clean structuring.
* **CSS3:** Custom variables, CSS flexbox/grid layout, and smooth animations.
* **JavaScript (ES6):** Dynamic DOM injection, state filtering, and LocalStorage data handling.
* **FontAwesome Icons:** Professional vector iconography for navbar and footer buttons.
* **Google Fonts:** Playfair Display (Titles) & Poppins (Body text).

---

## 📂 Folder Structure
* `index.html` / `collection.html` / `arrival.html` / `deal.html` / `about.html` — Layout files.
* `browser.js` — Core handling script for filtering catalog and cart operations.
* `browser.css` — Global styling theme sheet.
* `image/` — Local repository folder for product assets.
